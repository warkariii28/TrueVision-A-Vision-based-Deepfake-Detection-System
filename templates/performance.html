Classic Comparison Table

Model comparison with original metric names

This keeps the traditional project table view, while the notes below explain the terms in simpler words.

Accuracy: overall correctness.
Precision: how often a positive result is truly correct.
Recall: how many important cases the model catches.
F1 Score: a balanced score using precision and recall together.
AUC-ROC: overall ability to separate real and fake images.
PR-AUC: how well the model performs when precision and recall both matter.
